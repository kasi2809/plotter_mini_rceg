# Summary

<h1>Auto-Feed Soldering Station.</h1>
<i>Solution for extensive soldering or when a free hand is required.</i>

=========================================================

<b>ALPHA state!
Project is not complete!
Please comment anything about this project.</b>

Project Units
=========================================================
- Soldering station controller box
 - Electronis: Atmega + ULN2308

- Solder wire feeder engine
 - Motor : M42SP-4NP

- Solder Iron stand + sponge 

- Solder Iron dispencer unit
 - Guide tube : Disassembled AirTV Antenna.

- Tools stand #1 (Vertical tools) + screwdriver bits holder.

- Tools stand #2 (Horisontal tools) + trays

- [Foot switch](https://www.thingiverse.com/thing:3152310)

Problems
=========================================================
- Need make proper head for dispersion.. non metallic!
- Solder wire after 1-2 time usage... deforms and makes jams.. need make some non destructive mechanics.
- Very Noisy (like cheap plastic toys)!
- Block for solder Iron holding must be maded from other material.. not PLA 8) or must be painted heat proof paint. Partitally i solve this with split block to 3 parts,, and main heat reciver part maded from wood on CNC router.
- Need change motor to some 12v version what people can buy from aliexpress.
- Need moar magnets... i love magnets
